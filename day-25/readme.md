## 🎯 Objective

The objective of today's lesson was to understand how Machine Learning engineers automatically find the best combination of hyperparameters instead of manually guessing different values.

---

## 🧠 What I Learned

- Learned what Grid Search is and why it is used.
- Understood that manually selecting hyperparameters is based on guessing, while Grid Search systematically tests every possible combination.
- Learned how to create a parameter grid using a Python dictionary.
- Understood the role of `GridSearchCV` in automating hyperparameter tuning.
- Learned how to obtain the best hyperparameters using `best_params_`.
- Understood that Grid Search does not magically improve a model; it only finds the best combination among the values we provide.
- Learned that choosing good hyperparameters can improve model performance and generalization.
