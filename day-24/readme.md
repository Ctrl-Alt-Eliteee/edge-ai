## Key Learning

Today I learned that hyperparameters control how a model learns before training begins. The min_samples_split hyperparameter decides when a Decision Tree is allowed to split a node. Choosing values that are too small can lead to overfitting, while values that are too large can cause underfitting. The goal is to tune these values to achieve good generalization.
