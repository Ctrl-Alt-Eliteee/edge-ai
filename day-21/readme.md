# Day 21

## Topics Covered

- Reading a Confusion Matrix
- Rows vs Columns
- Correct Predictions vs Errors
- Error Analysis
- Choosing Evaluation Metrics Based on the Application

## Key Learning

Today I learned how to read a confusion matrix and understand exactly where a machine learning model makes mistakes. I also learned that the best evaluation metric depends on the application. In safety-critical systems like factories and hospitals, understanding the type of error is often more important than achieving the highest accuracy.
