Accuracy:


Out of all predictions, how many were correct?


Precision:


when the model predicts positive how often is it correct?


P = Tp/(TP+FP)
higher the precision --> few false positives


Recall:

of all actual positives how many did the model find?

R = TP/(TP+FN)

higher the recall --> few false negatives


F1 Score:

Precision and recall often trade off against each other.

The F1 score combines both into a single metric using the harmonic mean


F1=2 x (P x R)/(P + R)
