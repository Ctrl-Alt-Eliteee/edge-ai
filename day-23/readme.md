## Key Learning

Today I learned that machine learning models are not only trained—they are also tuned. Hyperparameters like max_depth are chosen before training and can significantly affect model performance. The goal is not to build the biggest model but to build one that generalizes well while balancing accuracy, complexity, and efficiency.
